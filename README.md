# movie_mate
영화 소개 밑 리뷰가능 사이트

## 프로젝트 소개
왓챠피디아를 참고하여 만든 영화 소개 밑 리뷰 사이트 입니다.

## 개발 기간
* 23.04.10 ~ 23.05.15
* 04.10 ~ 04.11 클론코딩 대상 결정
* 04.11 ~ 04.14 데이터베이스 스키마 설계 밑 html 전개도 구축
* 04.14 ~ 04.19 oepn api 기반 영화 데이터를 원하는 형태로 정제후 기본 데이터 베이스 구축 밑 스프링 베포
* 04.19 ~ 05.15 사이트 제작

## 개발 환경
- "Java 11"
- "JDK 1.11.0"
- "IDE : STS 3.7"
- "Database : Oracle DB (11xe)"
- "ORM : Mybatis"
